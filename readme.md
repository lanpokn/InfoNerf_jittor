主要改动为：
1.读取时不进行存储工作。可能本人所用的3050ti性能较差，多余的存储一定导致崩溃
2.读取ckpt的model文件指定文件名而不是文件夹
3.软件工程相关的大量优化，使得代码结构和文件结构更清晰
4.显示优化，可以实时得知迭代次数
5.删除除psnr外的metric，因为大作业并没有要求相关指标
